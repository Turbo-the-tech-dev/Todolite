## ğŸ“– How to Use

1. **Add a Task**: Type your task in the input field and click "Add Task" or press Enter
2. **Complete a Task**: Click on the task text to mark it as complete (or incomplete)
3. **Edit a Task**: Click the "Edit" button next to any task to modify its text
